# Leapflow

Your AI Agent to supercharge workflow

## Project structure

```bash
src
├── components
│   └── ui
├── config
├── lib
│   ├── hooks
│   ├── utils
├── styles
│   └── globals.css
├── types
```

## Getting Started

- supports both npm and bun

```bash
npm install
```

or
  
```bash
bun install
```

## Running the app

```bash
npm run dev
```

or

```bash
bun dev
```
